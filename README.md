## Logistics & Supply Chain Automation
Этот проект имитирует автоматизацию проверки входящих поставок (например, для дистрибуции FMCG).

**Функционал:**
- Обработка данных в формате JSON.
- Автоматическая проверка целостности данных.
- Расчет логистических KPI (оценка полноты поставки).
- Валидация условий хранения (температурный режим).
